计时器

功能实现

  1. 计时1.5小时
  2. 期间任意3-5分钟会播放叮的一声
  3. 播放结束后会播放一个提示音

环境

  • SDL_Mixer:支持wav格式,自己找两个替换bing.wav,over.wav,可以到这里找: https://freesound.org/

  • linux-ubuntu下载：sudo apt-get install libsdl2-mixer-dev

work.hpp

tool.cpp

编译

  • 编译：g++ tool.cpp -o work -lSDL2 -lSDL2_mixer
