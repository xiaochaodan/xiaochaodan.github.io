MySQL

发表于2025-04-22|更新于2025-04-25
|总字数:1k|阅读时长:3分钟|浏览量:

启动

  1. sudo service mysql status:查看mysql服务状态
  2. sudo service mysql start:启动mysql服务

第二章

  1. 登录数据库 mysql -u root -p
  2. 推出数据库 exit/quit

第三章

  1. USE 数据库名;选择数据库
  2. SHOW DATABASES; 显示数据库
  3. SHOW TABLES; 显示当前数据库的列表
  4. SHOW COLUMNS FROM 表名; 显示表字段
    1. DESCRIBE 表名; 显示表字段,等价与SHOW COLUMNS FROM 表名;
  5. SHOW STATUS; 显示服务器状态，给你一份健康报告
  6. SHOW CREATE DATABASE 数据库名; 显示创建数据库的SQL语句
  7. SHOW CREATE TABLE 表名; 显示创建表的SQL语句
  8. SHOW ERRORS; 显示上一次语句的错误
  9. SHOW WARNINGS; 显示上一次语句的警告

补充1

  1. CREATE DATABASE 数据库名; 创建数据库
  2. DROP DATABASE 数据库名; 删除数据库
  3. CREATE TABLE 表名 (字段名 字段类型 约束,
               字段名 字段类型 约束
           ...
    );
  4. INSERT INTO 表名 (字段名, 字段名, …) VALUES (值, 值, …); 插入数据
  5. ALTER TABLE 表名 ADD [COLUMN] 字段名 数据类型 [其他约束]; 添加字段([COLUMN],可选)
  6. UPDATE 表名 SET 列名 = 新值 WHERE 条件; 修改数据
    1.例子：
    单独： UPDATE products SET money = 20 WHERE id = 1;
         UPDATE products SET number = 0 WHERE number IS NULL;  修改值为0的number字段
    批量：
    UPDATE products
    SET money = CASE id
    WHEN 1 THEN 20
    WHEN 2 THEN 30
    ELSE money
    END;

📦 常见 SQL 数据类型

数据类型 描述
INT / INTEGER 整数类型，常用于 ID、数量等。可加 UNSIGNED 表示非负。
SMALLINT 占用更小空间的整数，适合范围小的值。
BIGINT 用于非常大的整数，如用户量巨大的系统。
FLOAT / DOUBLE 浮点数，适合温度、速度等有小数的值。
DECIMAL(p, s) 精确浮点数，常用于金融，p 为总位数，s 为小数位数。
CHAR(n) 定长字符串，固定为 n 个字符，适合身份证号等。
VARCHAR(n) 可变长度字符串，最多 n 个字符，适合姓名、地址等。
TEXT 长文本，适合存放文章、备注等内容。
DATE 日期类型，格式 YYYY-MM-DD。
TIME 时间类型，格式 HH:MM:SS。
DATETIME 日期 + 时间，格式 YYYY-MM-DD HH:MM:SS。
BOOLEAN 布尔类型，TRUE 或 FALSE，有些数据库用 1/0。

🧱 常见 SQL 约束

约束关键字 描述
PRIMARY KEY 主键，唯一标识每一行记录，不能重复也不能为空。
NOT NULL 不允许为空，必须有值。
UNIQUE 值必须唯一，不可重复。
DEFAULT 指定默认值，插入时未指定则使用该值。
CHECK (条件) 添加条件限制，数据必须满足该条件。
FOREIGN KEY 外键，用于关联其他表的主键，建立表之间的关系。

第四章 检索数据

  1. SELECT 字段名 FROM 表名; 检索一个或多个列
  2. SELECT * FROM 表名; 检索所有列
  3. SELECT DISTINCT 字段名 FROM 表名; 检索不同的列
  4. SELECT 字段名 FROM 表名 LIMIT 值; 检索指定数量的记录
    1. SELECT 字段名 FROM 表名 LIMIT 值 OFFSET 值; 检索指定数量的记录,从第几条开始，几条之后结束
    2. SELECT 字段名 FROM 表名 WHERE 条件 LIMIT 值,值 检索指定数量的记录,从第几条开始,几条后结束
  5. SELECT 表名.字段名 from 表名; 检索指定表的列

第五章 排序检索数据

  1. SELECT 字段名 FROM 表名 ORDER BY 字段名; 检索数据并排序(默认升序)
  2. SELECT 字段名 FROM 表名 ORDER BY 字段名 DESC; 检索数据并排序(降序)
  3. SELECT 字段名 FROM 表名 ORDER BY 字段名 ASC,字段名 DESC; 检索数据并排序(多个字段，一个字段一个排序方法)
  4. SELECT 字段名 FROM 表名 ORDER BY 字段名 DESC LIMIT 值; 检索数据并排序(指定数量)

第六章 过滤数据

  1. SELECT 字段名 FROM 表名 WHERE 条件; 检索满足条件的数据
    1. 条件：>，<，=，!=,<>(不等于),>=,<=
    2. SELECT 字段名 FROM 表名 WHERE 字段名 BETWEEN 值1 AND 值2; 检索指定范围内的数据
  2. SELECT 字段名 FROM 表名 WHERE 字段名 IS NULL; 检索为空的数据
